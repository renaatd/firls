FIR LS

FIR LS is a free tool to design symmetric FIR filters using the least squares method. It designs type I and type II symmetrical FIR filters.

The implementation is based on SciPy signal.firls. The algorithm has been translated to C++, extended for type II FIR filters, and compiled to WebAssembly using Emscripten.
